bash
# 创建虚拟环境(推荐)
python -m venv venv

# 确保是 kirara-ai 运行时环境
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

# 安装 Onebot Adapters
pip install chatgpt-mirai-qq-bot-onebot-adapter

# 启动 kirara-ai
pip -m kirara-ai

应用插件 ⚙️ ​

  • 安装完成后, 进入插件管理->插件列表->启用im_onebot_adapters
  • 也可以在插件管理->已安装插件->启用im_onebot_adapters
